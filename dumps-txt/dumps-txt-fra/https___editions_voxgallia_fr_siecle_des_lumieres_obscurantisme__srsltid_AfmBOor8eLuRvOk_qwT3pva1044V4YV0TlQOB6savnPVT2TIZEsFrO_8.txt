Le siècle des Lumières : mouvement philosophique et littéraire anticatholique



Au XVIIIe siècle, plus précisément entre 1715, année de la mort de Louis XIV, et 1789, année marquée par la Révolution française, émergea un mouvement philosophique et littéraire connu sous le nom des « Lumières ». Les penseurs et savants qui en faisaient partie avaient pour ambition de remettre en question les fondements religieux, politiques, économiques et sociaux de la monarchie multiséculaire.



L'Encyclopédie de Diderot et d'Alembert, publiée entre 1751 et 1772, devint l’œuvre emblématique de cette contestation. Visant à rassembler et diffuser le savoir de l'époque, cette œuvre fut initialement bien accueillie par l'Église et la monarchie. Cependant, les tensions montèrent lorsque ces institutions comprirent que l'Encyclopédie servait en réalité d'outil politique contre elles, avec les philosophes pour adversaires.






Mais que représente réellement cette idéologie des Lumières, et quels en furent les effets concrets ? À cette époque, les savants et philosophes se fixaient pour mission de combattre ce qu'ils appelaient l'obscurantisme et l'absolutisme, soit le catholicisme et la monarchie. Cette démarche aboutit à la Révolution de 1789, qui entraîna massacres et violences, comme en Vendée. Cette idéologie a-t-elle vraiment apporté l'émancipation qu'on lui attribue, ou ne fut-elle qu'un prétexte pour abattre la monarchie capétienne au profit des puissances financières ? Lorsque l'on observe la France actuelle, il est légitime de s'interroger sur l'efficacité de cette idéologie en termes de progrès et de libération des peuples.



Par ailleurs, en lisant les œuvres de figures majeures comme Jean-Jacques Rousseau et Voltaire, il est frappant de constater que ces soi-disant « éclairés » avaient des vues régressives sur les femmes, les Noirs, les Juifs, et même sur le peuple en général.



Cela étant, nous préconisons de rejeter ce qui est à rejeter dans l’idéologie des Lumières et de ne retenir que ce qui est véritablement positif, selon une approche empirique.



Pour nous, royalistes, les Lumières ne constitueront jamais le socle de nos valeurs. Nous puisons nos références dans des sources plus fiables, comme les Évangiles et l'éthique chevaleresque, que nous vous invitons à découvrir ou redécouvrir. Au « siècle des Lumières », nous préférons la lumière des siècles !

            // off canvas
            function oxygen_offcanvas_menu_init(menu) {

                // only init off-canvas animation if trigger icon is visible i.e. mobile menu in action
                var offCanvasActive = jQuery(menu).siblings('.oxy-pro-menu-off-canvas-trigger').css('display');
                if (offCanvasActive!=='none') {
                    var animation = menu.data('oxy-pro-menu-off-canvas-animation');
                    setTimeout(function() {menu.attr('data-aos', animation);}, 10);
                }
                else {
                    // remove AOS
                    menu.attr('data-aos', '');
                };
            }
            
            function oxygen_offcanvas_menu_run(menu) {

                var container = menu.find(".oxy-pro-menu-container");
                
                if (!container.attr('data-aos')) {
                    // initialize animation
                    setTimeout(function() {oxygen_offcanvas_menu_toggle(menu, container)}, 0);
                }
                else {
                    oxygen_offcanvas_menu_toggle(menu, container);
                }
            }

            var oxygen_offcanvas_menu_toggle_in_progress = false;

            function oxygen_offcanvas_menu_toggle(menu, container) {

                if (oxygen_offcanvas_menu_toggle_in_progress) {
                    return;
                }

                container.toggleClass('aos-animate');

                if (container.hasClass('oxy-pro-menu-off-canvas-container')) {
                    
                    oxygen_offcanvas_menu_toggle_in_progress = true;
                    
                    var animation = container.data('oxy-pro-menu-off-canvas-animation'),
                        timeout = container.data('aos-duration');

                    if (!animation){
                        timeout = 0;
                    }

                    setTimeout(function() {
                        container.removeClass('oxy-pro-menu-off-canvas-container')
                        menu.removeClass('oxy-pro-menu-off-canvas');
                        oxygen_offcanvas_menu_toggle_in_progress = false;
                    }, timeout);
                }
                else {
                    container.addClass('oxy-pro-menu-off-canvas-container');
                    menu.addClass('oxy-pro-menu-off-canvas');
                    oxygen_pro_menu_set_static_width(menu);
                }
            }

            function oxygen_init_search($) {

                  
                $('body').on( 'click', '.oxy-header-search_toggle', function(e) {           
                        e.preventDefault();
                        let $toggle = $(this);
                        let $form = $toggle.closest('.oxy-header-search').find('.oxy-header-search_form');
                        
                        
                        if (!$form.hasClass('visible')) {
                            showSearch($toggle);
                            
                        } else {
                            hideSearch($toggle);
                        }
                    }
                );
                
                // Tabbing out will close search
                $('.oxy-header-search_toggle').next('.oxy-header-search_form').find('input[type=search]').on('keydown', function (event) {
                    
                    let togglebutton = $('.oxy-header-search_toggle');

                    if (event.keyCode === 9) {
                      hideSearch(togglebutton);
                    }

                });
                
                // Pressing ESC will close search
                $('.oxy-header-search_toggle').next('.oxy-header-search_form').find('input[type=search]').keyup(function(e){
                    
                    let togglebutton = $('.oxy-header-search_toggle');
                    if(e.keyCode === 27) {
                      hideSearch(togglebutton);
                    } 
                  });
                
                
                // Helper function to show the search form.
                function showSearch(toggle) {
                    
                    toggle.closest('.oxy-header-search').find('.oxy-header-search_form').addClass('visible');

                    if (true === toggle.closest('.oxy-header-search').find('.oxy-header-search_toggle-open').data('prevent-scroll')) {
                        $('html,body').addClass('oxy-header-search_prevent-scroll');
                    }

                    setTimeout(
                    function() {
                        toggle.closest('.oxy-header-search').find('input[type=search]').focus();
                    }, 300);

                }

                // Helper function to hide the search form.
                function hideSearch(toggle) {

                    toggle.closest('.oxy-header-search').find('.oxy-header-search_form').removeClass('visible');
                    $('html,body').removeClass('oxy-header-search_prevent-scroll');

                    setTimeout(
                    function() {
                    toggle.closest('.oxy-header-search').find('.oxy-header-search_toggle-open').focus();
                    }, 0);

                }

             };

            // Initialize Oxygen Modals
            jQuery(document).ready(function() {

                function showModal( modal ) {
                    var $modal = jQuery( modal );
                    $modal.addClass("live");
                    var modalId = $modal[0].querySelector('.ct-modal').id;
                    
                    var focusable = modal.querySelector('a[href]:not([disabled]), button:not([disabled]), textarea:not([disabled]), input[type="text"]:not([disabled]), input[type="radio"]:not([disabled]), input[type="checkbox"]:not([disabled]), select:not([disabled])');

                    if(focusable) {
                        setTimeout(() => {
                            focusable.focus();    
                        }, 500);
                    } else {
                        setTimeout(() => {
                        $modal.focus();
                        }, 500)
                    }

                    // Check if this modal can be shown according to settings and last shown time
                    // Current and last time in milliseconds
                    var currentTime = new Date().getTime();
                    var lastShownTime = localStorage && localStorage['oxy-' + modalId + '-last-shown-time'] ? JSON.parse( localStorage['oxy-' + modalId + '-last-shown-time'] ) : false;
                    // manual triggers aren't affected by last shown time
                    if( $modal.data( 'trigger' ) != 'user_clicks_element' ) {
                        switch( $modal.data( 'open-again' ) ) {
                            case 'never_show_again':
                                // if it was shown at least once, don't show it again
                                if( lastShownTime !== false ) return;
                                break;
                            case 'show_again_after':
                                var settingDays = parseInt( $modal.data( 'open-again-after-days' ) );
                                var actualDays = ( currentTime - lastShownTime ) / ( 60*60*24*1000 );
                                if( actualDays < settingDays ) return;
                                break;
                            default:
                                //always show
                                break;
                        }
                    }

                    // Body manipulation to prevent scrolling while modal is active, and maintain scroll position.
